Weekly Nerd - Week 1 - Kilian Valkhof

Kilian Valkhof werkt aan de developer vriendelijke browser Polypane. Hij is een voorstander van het gebruik van CSS boven bijvoorbeeld Javascript. Dit doet hij aan de hand van de "rule of least power". Dit houdt in dat je het beste de "leas powerfull" programeertaal kan gebruiken.

Hij heeft vooral uitleg gegeven over de verschillende dingen die mogelijk zijn met alleen Css waar veel developers nog Javascript voor gebruiken. Dit is niet gewenst omdat Javascript meet processing power vraagt.

Deze dingen waren:

  • Custom toggles
  • Datalists
  • Colorpickers
  • In-Page transitions
  • Smooth scrolling
  • Carousels
  • Accordions and models
  • Dialogs

Doordat ik zelf nog weinig weet van Javascript ga ik altijd eerst op zoek naar een oplossing met alleen Html en Css, pas als dit niet blijkt te kunnen met deze talen ga ik kijken naar Javascript. Ik vond dit dus een interessante weekly nerd om naar te luisteren.

Weekly Nerd - Week 2 - Fenna de Wilde

Fenna de Wilde noemt zichzelf een "creative developer". Dit kwam doordat ze werkt als developer maar ook graag beslissingen maakt over designs en derglijke. Ze liet zien dat ze gewerkt heeft aan Phantom en dat ze werkzaam is bij het bedrijf Bakkenbeack.

Ze wilde het vooral hebben over de toegankelijkheid van websites. Dit houdt in dat zoveel mogelijk mensen je website kunnen gebruiken. Vooral kijk je naar hoe screenreaders reaggeren op je website want dit is erg belangrijk voor veel mensen met een beperking.

Dit waren de onderwerpen die zijn besproken:

  • Focus states
  • Aria labels
  • Toegankelijkde carousels
  • Focus guards
  • Kleurcontrast
  • Fontsizes en regellengtes
  • Alt tekst
  • Skip to content button
  • Reduced motion
  • Autoplay videos toggles

Heel eerlijk gezegd vind ik dit niet het meest interessante onderwerp. Wel begrijp ik goed waarom dit belangrijk is. Ik heb met eerdere vankken op CMD veel meegekregen over toegankelijkheid dus veel dingen die ik hier heb gehoord kende ik al. Ook heb ik wat nieuwe dingen geleerd die ik ongetwijfeld ga gebruiken bij latere projecten.
